# גאומטריה — לומדים חכם

אפליקציית לימוד גאומטריה בעברית לרמת חטיבת ביניים ותיכון.

## הרצה

```powershell
powershell -ExecutionPolicy Bypass -File .claude/serve.ps1
```

פתח בדפדפן: `http://localhost:3333`

> האפליקציה לא עובדת מ-`file://` — יש להפעיל דרך שרת.

## תוכן

| קטגוריה | מספר משפטים |
|---|---|
| זוויות וישרים מקבילים | 5 |
| משולשים | 40 |
| מרובעים | 29 |
| מעגלים | 27 |
| **סה"כ** | **101 משפטים** |

בנוסף: **36 הגדרות** עם ניווט מלא ושרטוטים.

## תכונות

- ניווט hash-based — כל מסך בעל כתובת URL משלו, כפתור "חזרה" עובד
- שרטוטים גיאומטריים SVG לכ-100 משפטים
- חיפוש הגדרות ופתיחה במודל
- עיצוב RTL עברי מלא
- מותאם למובייל

## מחסנית טכנית

HTML + CSS + Vanilla JavaScript בלבד — ללא frameworks, ללא bundler, ללא תלויות חיצוניות.
